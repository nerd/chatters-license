# Chatters' License

Chatters' License is a lightweight and highly compatible Minecraft mod that allows players to bypass account-level restrictions.

## Compatibility

This mod has explicitly been tested on the following versions:

- 1.16.4
- 1.17.1
- 1.18.2
- 1.19.4
- 1.20.6
- 1.21.11
- 26.1-snapshot-9

## Dependencies

- Fabric Loader
